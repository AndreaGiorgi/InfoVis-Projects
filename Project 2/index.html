Radiation level monitoring became of major importance after the Chernobyl events of 1986 and more so after the Fukushima events of 2011. The Norwegian government, as well as the Swedish, Danish and Finnish governments, monitors radiation levels throughout the country on a daily basis throughout various monitoring stations. Attention to this issue has increased starting from 22/23 June 2020, when monitoring stations detected abnormal levels of several radiactive elements like cesium-134, cesium-137, cobalt-60 and ruthenium-103 (source) in the North and Norwegian sea.

The following set of visualizations utilize radiation data derived from DSA Norwegian Radiation and Nuclear Safety Authority. Each daily data has been aggregated by each county
showing the daily radiation change on a two week range of observation. The data is taken
directly from the DSA Open Data, carefully engineered and later visualized.

For more information about the data used in this visualization project, the data is stored on Github alongside with the source code.
